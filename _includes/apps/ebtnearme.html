EBT Near Me (Alpha)

EBT Near Me shows the nearest places you can use an EBT card nationwide.

Created by

  • Rhys Fureigh
  • Alan Williams

EBT Near Me shows the nearest places you can use an EBT card nationwide. We built it because the current tools leave something to be desired and @allafarce inspired us with BART me now.

FAQ

How did it start?
It started when one of our teammates enrolled in CalFresh (Food Stamps) and found it tricky to figure out where he could use them. Check out the project ideas repository on GitHub to see the beginning conversation.

What does Alpha mean?
Alpha means you shouldn't count on it working tomorrow. This is an experiment for now. Once it's stable enough to use we'll upgrade move it to Beta.

How does it work?
We download data from FNS and put on a map with CartoDB. See the source code for details.
